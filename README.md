# Mudanzas Metepec · Sitio web

Sitio web profesional enfocado en **conversión** para Mudanzas Metepec — mudanzas, fletes y transporte en Metepec, Toluca, Estado de México y toda la República Mexicana.

> **Slogan:** Conoce el arte de mudarte
> **WhatsApp / Tel:** 722 830 0083

## 🎯 Objetivo
Generar más **cotizaciones por WhatsApp**, llamadas y solicitudes de contacto. Cada sección está pensada para generar confianza, reducir objeciones y guiar al visitante a contactar.

## 📁 Estructura
```
index.html      → Página principal (one-page) con todas las secciones
styles.css      → Estilos y sistema de diseño (paleta derivada del logo)
script.js       → Menú móvil, formulario→WhatsApp, contadores y animaciones
assets/img/     → Logo (agrega aquí tus fotos reales)
```

## ✨ Secciones incluidas
Header fijo · Hero con cotizador · Barra de confianza · Diferenciadores ·
Problema/Solución · Servicios · Planes (Básico/Intermedio/Premium) · Proceso ·
Seguridad (cámaras corporales) · Cobertura nacional · Opiniones reales ·
Galería · Formulario de cotización (abre WhatsApp prellenado) · 16 preguntas
frecuentes · CTA final · Footer · Botón flotante de WhatsApp · Barra móvil de contacto.

## 🚀 Cómo verlo
Abre `index.html` en tu navegador. No requiere build ni dependencias.

Para servirlo localmente:
```bash
python3 -m http.server 8000
# luego visita http://localhost:8000
```

## 🔧 Personalización rápida
- **Número de WhatsApp:** está en `index.html` (enlaces `wa.me/527228300083`) y en `script.js` (`WHATSAPP_NUMBER`).
- **Fotos reales:** sustituye los bloques `.gallery__item` en `index.html` por `<img>` con tus fotografías dentro de `assets/img/`.
- **Dominio:** actualiza las URLs `https://www.mudanzasmetepec.com/` (canonical, Open Graph y Schema.org) cuando tengas el dominio definitivo.

## 🔎 SEO
Incluye meta title/description, Open Graph, Twitter Cards y datos estructurados
**Schema.org `MovingCompany`** (calificación, servicios, cobertura y horario),
optimizado para keywords de mudanzas y fletes en Metepec, Toluca, Estado de
México, CDMX, Querétaro, Puebla y a nivel nacional.
