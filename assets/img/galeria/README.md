# Galería de fotos · Mudanzas Metepec

Coloca aquí las fotografías reales que se mostrarán en la sección **Galería** del sitio.

## Cómo usarlas
1. Guarda cada imagen en esta carpeta con un nombre descriptivo. Nombres sugeridos
   (coinciden con los comentarios en `index.html`):
   - `01-unidad-cerrada.jpg`
   - `02-emplaye-profesional.jpg`
   - `03-muebles-protegidos.jpg`
   - `04-maniobras.jpg`
   - `05-transporte-especial.jpg`
   - `06-entrega-destino.jpg`
2. En `index.html`, dentro de cada `<figure class="gallery__item">`, descomenta la
   línea `<img>` y borra el placeholder (emoji + `<span>`).

## Recomendaciones técnicas
- **Tamaño:** ~1200 × 900 px (proporción 4:3).
- **Formato:** `.jpg` o `.webp` (mejor compresión).
- **Peso:** idealmente menos de 300 KB por imagen para que el sitio cargue rápido.
- **Texto alt:** describe la foto pensando en SEO (ej. "Mudanza residencial en Metepec con muebles emplayados").
